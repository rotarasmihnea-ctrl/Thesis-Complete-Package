# Thesis Code: LLM Alignment with Human Judgments

This repository contains the code used for my bachelor thesis on evaluating the alignment between large language models and human judgments of contextual dialogue relevance.

## Contents
- Data preprocessing
- Baseline condition evaluation
- Theory-of-Mind condition evaluation
- Statistical analysis (correlation, MSE, regression, classification)
